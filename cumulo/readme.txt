Welcome to Cumulo - Responsive Multi-Purpose WordPress Theme !

Cumulo is a minimal looking yet feature-rich WordPress theme you can use for any type of website.
It is clean, responsive and super flexible. Based on Visual Composer and comes with two premium sliders - Slider Revolutiona and LayerSlider.
Cumulo is very scalable and offers hundreds customization options you can use to build your corporate landing pages to small/medium studios and personal pages. 
Born with 16 unique home demos, more will be added in no time.
